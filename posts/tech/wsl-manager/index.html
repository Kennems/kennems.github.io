wslеҲқе§ӢеҢ–#

еҲӣе»әж–°з”ЁжҲ·#

дҪҝз”Ёд»ҘдёӢе‘Ҫд»ӨеҲӣе»әеҗҚдёә zg зҡ„ж–°з”ЁжҲ·пјҡ

sudo adduser zg
  • зі»з»ҹдјҡжҸҗзӨәдҪ иҫ“е…Ҙ ж–°еҜҶз Ғ е’Ң зЎ®и®ӨеҜҶз ҒгҖӮ
  • жҺҘдёӢжқҘпјҢдҪ иҝҳдјҡиў«иҰҒжұӮеЎ«еҶҷдёҖдәӣйўқеӨ–дҝЎжҒҜпјҲеҰӮе…ЁеҗҚгҖҒжҲҝй—ҙеҸ·зӯүпјүпјҢиҝҷдәӣеҸҜд»ҘжҢүйңҖеЎ«еҶҷпјҢд№ҹеҸҜд»ҘзӣҙжҺҘжҢү Enter и·іиҝҮгҖӮ

е°Ҷз”ЁжҲ·ж·»еҠ еҲ° sudo з»„пјҲеҸҜйҖүпјү#

еҰӮжһңдҪ еёҢжңӣ zg з”ЁжҲ·е…·жңүз®ЎзҗҶе‘ҳжқғйҷҗпјҢеҸҜд»Ҙе°Ҷе…¶ж·»еҠ еҲ° sudo з»„пјҡ

sudo usermod -aG sudo zg

еҲҮжҚўеҲ°ж–°з”ЁжҲ·#

е®ҢжҲҗеҲӣе»әеҗҺпјҢдҪ еҸҜд»ҘдҪҝз”Ёд»ҘдёӢе‘Ҫд»ӨеҲҮжҚўеҲ° zg з”ЁжҲ·пјҡ

su - zg

жҲ–иҖ…дҪҝз”Ёпјҡ

sudo -i -u zg

и®ҫзҪ®еҜҶз ҒпјҲеҰӮжһңдҪ йңҖиҰҒжӣҙж”№еҜҶз Ғпјү#

еҰӮжһңдҪ йңҖиҰҒдёәз”ЁжҲ·и®ҫзҪ®жҲ–жӣҙж”№еҜҶз ҒпјҢеҸҜд»ҘдҪҝз”Ёд»ҘдёӢе‘Ҫд»Өпјҡ

sudo passwd zg

з„¶еҗҺиҫ“е…Ҙж–°еҜҶз ҒгҖӮ

жҹҘзңӢз”ЁжҲ·еҲ—иЎЁпјҲйӘҢиҜҒпјү#

дҪ еҸҜд»ҘйҖҡиҝҮд»ҘдёӢе‘Ҫд»ӨжҹҘзңӢеҪ“еүҚзі»з»ҹдёӯзҡ„жүҖжңүз”ЁжҲ·пјҡ

cat /etc/passwd

йҖҖеҮәеҪ“еүҚз”ЁжҲ·дјҡиҜқ#

е®ҢжҲҗеҗҺпјҢеҰӮжһңдҪ иҰҒйҖҖеҮәеҪ“еүҚз”ЁжҲ·дјҡиҜқпјҢеҸҜд»ҘдҪҝз”Ёпјҡ

exit
