Android面试相关

Android面试相关 1.Activity 说下Activity生命周期 Activity 有 7 个核心生命周期回调： onCreate() — 系统首次创建 Activity 时调用，进行基础初始化（绑定数据、关联 ViewModel、实例化变量） onStart() — Activity 进入"已启动"状态，对用户可见 onResume() — Activity 进入前台，可与用户交互 onPause() — 用户离开 Activity 的第一个信号，Activity 不再处于前台但仍可能可见（多窗口模式）。执行非常短暂，不要做耗时操作 onStop() — Activity 不再可见，可执行较重的关闭操作（如保存数据到数据库） onRestart() — 已停止的 Activity 重新被带回前台时调用（在 onStop 和 onStart 之间） onDestroy() — Activity 被销毁前调用，释放所有资源 生命周期流程：Created → Started → Resumed → Paused → Stopped → Destroyed 从停止状态恢复：Stopped → onRestart() → Started → Resumed ...
